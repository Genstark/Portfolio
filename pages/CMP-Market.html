Project Overview

At Compro Technologies, I developed CMP Market a responsive e-commerce application with React on the frontend and Node.js + Express on the backend, using MongoDB as the database. It features role-based access control, secure authentication, inventory management, and real-time data processing. After completion, I successfully sold this project, showcasing its market viability.
