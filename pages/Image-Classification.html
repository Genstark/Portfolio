Project Overview

Developed an image classification model capable of categorizing objects such as fruits, vegetables, humans, and animals based on user-uploaded images. Using a convolutional neural network (CNN), the model's accuracy was enhanced through image preprocessing, hyperparameter tuning, and overfitting prevention. A user-friendly interface was created for real-time uploads and predictions, significantly refining expertise in machine learning and image processing, especially in CNNs and model optimization.

Technologies Used

  • Python
  • Tensorflow
  • Keras
  • OpenCV
