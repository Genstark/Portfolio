One Shot Bird Knockout

A precision 2D game where you have exactly one shot to knock out an entire group of birds. No second chances.

Overview

Project Overview

One Shot is a challenging 2D physics game where the player must use a single projectile to knock out all birds on screen. The game requires careful aiming and understanding of physics mechanics. Built with Python and Pygame, it features custom sprites, collision detection, and progressively harder levels.
