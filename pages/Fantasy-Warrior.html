Fantasy Warrior

Overview

Project Overview

Fantasy Warrior is a 2-player fighting game where each player selects a unique character with special abilities and powers. Built with Python and Pygame, the game features custom character animations, health bars, special moves, and dynamic combat mechanics. Each warrior has a distinct playstyle, encouraging strategic gameplay.
