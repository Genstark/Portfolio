Project Overview

Fantasy Warrior is an engaging multiplayer game where two users battle each other using unique characters with distinct powers and abilities. The game emphasizes strategy, skill, and timing, allowing players to outwit their opponents and emerge victorious. Each character has unique strengths and weaknesses, making every match dynamic and exciting. Built using Python and PyGame, Fantasy Warrior showcases the potential of these technologies in creating interactive and visually appealing games.
