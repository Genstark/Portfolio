Squash Game

A 2D game built in Godot Engine where the player collects coins and overcomes enemies across multiple levels.

Overview

Project Overview

Squash Game is a 2D game built using Godot Engine and GDScript. The player navigates through levels, collecting coins while avoiding and defeating enemies. The game features smooth character movement, animated sprites, a scoring system, and multiple levels. Game-over and win screens add a polished feel to the overall experience.

Technologies Used

  • Godot Engine
  • GDScript
  • Sprite Animation
  • Collision Detection
