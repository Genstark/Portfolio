Project Overview

Created a model to detect mask usage among students in educational environments, using computer vision and machine learning to identify non-compliance in real-time. The model analyzes video feeds to detect masks, recording details of violations for administrative follow-up. The system enhances safety measures by ensuring compliance with health protocols, while also generating reports and alerts for staff. This project refined expertise in machine learning, computer vision, and real-time data processing.

Technologies Used

  • Python
  • Tensorflow
  • Keras
  • OpenCV
