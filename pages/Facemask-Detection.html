Real-time Facemask Detection

A computer vision model that detects mask usage in real-time to enforce safety protocols in educational environments.

Overview

Project Overview

This real-time facemask detection system was developed to ensure safety compliance in educational institutions. Using deep learning and OpenCV, the model processes live webcam feeds to detect whether individuals are wearing masks, providing instant visual alerts for violations. The system achieves high accuracy even in varied lighting conditions.

Technologies Used

  • Python
  • TensorFlow
  • Keras
  • OpenCV
  • CNN
