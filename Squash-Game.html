Project Overview

Developed a 2D game featuring a character named Steve, whose mission is to collect coins while dodging and overcoming enemies. The project emphasized creating engaging game mechanics, with thoughtfully designed enemy behaviors and coin placements that encourage strategic play. Focused on balancing challenge and enjoyment, enhancing player engagement through intuitive controls, visual feedback, and sound effects, strengthening skills in game design, player interaction, and crafting an immersive, replayable gaming environment.

Technologies Used

  • Godot Engine
  • GD Script
