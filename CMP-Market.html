Project Overview

At Compro Technologies, I developed an e-commerce application named CMP Market using React for the front end and Node.js with Express.js for the back end, with MongoDB as the database. This project focused on creating a responsive, user-friendly interface that simplifies navigation, product browsing, and secure authentication for customers. On the back end, I designed a robust system with efficient data handling and role-based access control, allowing administrators to manage inventory and orders securely. After completion, I successfully sold this project, showcasing its value and viability in the market. Working on CMP Market enhanced my skills in building intuitive interfaces, securing user data, and optimizing performance for real-time data processing, providing a strong foundation in full-stack development.
